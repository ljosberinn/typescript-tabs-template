# Get started

```sh
git glone https://github.com/ljosberinn/typescript-tabs-template
cd typescript-tabs-template
npm install
```

# Adjustments

Default triggered class: `active-tab` (for adjustments, change `src/ts/tabs.ts` and `src/scss.tabs.scss`

# Dependencies
- Typescript `^2.9.2`
